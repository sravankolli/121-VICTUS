A hackathon project built by Team VICTUS to make working with PDFs smarter and easier.  
This tool extracts text from PDF files, runs AI-powered analysis (summarization, keyword extraction, Q&A), and presents insights in a simple web interface.

---
 Features
- Upload PDF files and extract text
- AI-powered summarization of content
- Keyword/topic extraction
- Ask questions directly about the PDF (Q&A)
- Metadata extraction (author, page count, etc.)
- Simple React frontend for interaction

Use of the PDF File Analyser
- Saves time: Instead of manually reading long PDFs, users can get quick summaries and insights.
- Makes documents interactive: You can ask questions about the PDF and get answers instantly.
- Helps in research/work: Useful for students, professionals, or anyone dealing with reports, contracts, or papers.
- Organizes information: Extracts keywords, topics, and metadata to make content easier to navigate.
